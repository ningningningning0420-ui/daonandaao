## 锁定结论（勿重新论证 / 勿回退）
- 引擎 = **账本 + 软护栏**；裁判 = 方向与内容；脾性留 prompt；裁决 = **单 call · verdict-first**。
- ★**立场 = 独立存储 · 两轴**（轴 A 留你 ↔ 放你走 / 轴 B 随你 ↔ 离你），和「情」**无公式关系**；裁判按个刃角度叙事性挪存值、引擎只读值出 tell。（早期「引擎公式现算倾向」已废——**勿回退**。）
- 一台 **tell 引擎**统起 留逃 / 因果 / 溯行军；溯行军 = 立场被锁死的**反常离群点**、外来非本丸所出。
- 重连 = **三道 gate**：认回 → 契满 → 重结契（= 认回 ∧ 契满 ∧ 轴 A 放你走 ∧ 轴 B 随你走）。
- **明 / 暗双线平行**：明线 = 治国（地基）、**暗线 = 四程（觉醒 / 重连 / 留逃 / 溯行军）= 主线**；焊点 = §3.3 每裁决三盘；暗线不给进度条（靠 tell 感）。
- **推手非死轨**（§17.0 开放总则）：创意 defer/transfer + 治理出色 + 平衡钳制 三路可度；失衡监测的牙挂民心 / 稳定 / 派系，不挂「照办没」。
- **删读心**：无免费明牌机制；行为 tell 只在叙事自然流露（免费），主动探全归 **情报系统**（花钱 / roll / 派系化 · 主导失守 → 真但片面 · 非金手指）。
- **世界绝大多数是普通 NPC**（软态）；6 刀 + 溯行军是人海里零星几根针，刀坐镇 / 傅役 / 妃皆特例非默认。
- **储君 = 人**：政治倾向（账本载重 dial）+ 三观「等等角度」（软态裁判养）；人格由五源合熬，**将军本人的行动是大头**。
