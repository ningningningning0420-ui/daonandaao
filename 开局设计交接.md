## 锁定红线(既有不变量 · 不许违反)
- **开局刀数固定 6**(schema C = 6 刀 + 1 溯行军 · §21 岔口 ③)。
- **L2 = 开局牌面**:无嗣无子未纳妃 · 立嗣三连在催婚阶 · 不预设中后期元素(§19.2)。
- **将军性别玩家选**(女 = 女性向 / 男 = BL · 刀妃恒男 · §0)。
- **世界线禁令**:刀连接成功前自报真名 = 被抹杀(§2)· onboarding 要立此禁。
- **RNG = hash 派生**(单 master seed 存信封 meta · 开局 roll〔L2 五维 + 溯行军 顶谁 / 暗堕原因〕走它 · §21 岔口 ③)。
- **引擎方向不回退**(§21 全锁 · onboarding 只「种」初始态、不改引擎)。
- **prompt 以 Klaude dramatron 为范式**(序章 / 问卷的 LLM 部分 · 神隐挑战 prompt 不参考)。
- **协作 feedback 红线**:别塞机械配额;给世界观事实让 AI 收敛;{{user}} 不写死(将军 persona 动态);prompt 留核心骨架、删可推导的。
