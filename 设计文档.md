## 0. 定位与基调

- 刀剑乱舞同人。**策略（经营 + 权谋）为骨架 + 恋爱为高权重支线 + 解密**，由 LLM 演算与扮演。
- **取向**：将军性别由玩家选定（女 / 男皆可，不写死）——女将军 ＝ 女性向（搭男性刀妃），男将军 ＝ BL；刀妃恒为男。世界仍是逆转大奥设定。
- 体验一句话：**白天治国，余光里周旋于一群暗中争夺你灵魂的爱人。**
- 分工总纲：**策略是地基**（决定你会不会死、哪些结局对你开放）；**恋爱/关系是剧情真正发生的地方**（决定你重连了谁、走不走得成、最终成为谁）。

## 1. 世界观与前提

- 你是**审神者**，带一队刀剑男士出任务时，坠入一条「不正确」的架空时间线——一个 **逆转大奥** 的世界（因为「不正确」所以架空，不必考虑历史还原度）。
- 世界线赋予你 **将军** 的身份。
  - **将军性别由玩家选定（女 / 男皆可）**，不写死；姓名、性格交给 persona。（女将军 = 女性向，男将军 = BL；刀妃恒为男。）
- 你的刀被赋予此世身份：
  - **妃子**（在此世是纯粹的人身，可与将军生育）；
  - **前朝官员**（影响策略层）。
  - 将军**可纳任何人**，但有对应的政治后果（见 §7）。
- **这个世界绝大多数是普通人**：朝堂百官、后宫诸妃、师傅乳母、市井黎民……多为此世土生的 **普通 NPC**（软态、裁判叙述，不进账本）。你的 **6 把刀（+ 混入的溯行军）是这片人海里零星几根针**，顶着此世身份散在其中——**你未必认得出谁是谁**（这正是 §2「认回我的刀」解密的底盘）。∴「刀坐镇某派 / 刀当傅役 / 刀妃」皆为 **特例**（你刻意安插已认出的刀，或一把尚未识破的刀早已在位），绝非各位置的默认。
- **生育是可选内容**：开局选定「由谁孕育」，锁为世界观、中途不变——女将军可「将军亲孕」或「刀妃孕」；男将军走「刀妃孕」（架空纯人身 + 世界线之力）或 过继 / 不生。
- **「失忆」的处理**：不能真让玩家失忆，所以剧情中你**清楚自己是审神者**，但**对这个世界一无所知**，需要刀来引路。
  - ∴ 真正悬而未揭的不是「我是不是审神者」，而是：谁是谁、谁是溯行军、出得去吗、代价是什么、我爱的是真是假。
- **觉醒 = 重新接上与刀的主从羁绊**（认回他们、以审神者之名重结契、力量归位）。觉醒够了，**回家的门才开**（详见 §6、§12）。

## 2. 开局（问卷 + 序章）

### 问卷
- 开局是一份 **问卷**：玩家填写自己的本丸情况 → 系统据此 **分析、定下随行的 6 把刀**（你交代的是「我是谁、我的本丸什么样」，而非列表点选）。
  - **呈现形式（meta）**：本质是问卷，实现上做成 **刀剑乱舞的「登录 → 配队 → 签署出阵命令」** 等更 meta 的界面——你是在「玩刀乱」的过程里被卷进去的。
- 问卷涵盖（够选刀 + 埋溯行军 + 立 persona + 调味世界）：
  - **你的审神者**：当审神者多久了？带队作风（温和 / 严格 / 疏离…）？→ persona + 刀如何待你。
  - **你的本丸**：什么气象？哪几把刀最得力 / 最亲近？与他们的渊源？→ 选刀池 + 溯行军候选 + 好感基线。
  - **生育倾向**：将军生 / 刀男生 / 不生 → 锁世界观开关。
  - **近侍是谁** → 随行刀的情感锚。
- 系统据此 **分析选 6 刀**。**溯行军另算，且一开始就不是你的刀**：它是一把**外来 / 敌方的暗堕刀**，在 6 刀定下后顶着某刀的身份（可与你某把刀同名 = doppelganger）**混入**你身边（详见 §10）。

### 序章
- 一小段 **审神者序章**：你点齐本丸、出阵（素材取自问卷）→ 意外坠入 → 醒来已是对此世无知的将军，身边已有一把刀在引你。
- 序章 **演足出阵的本丸**（让玩家对 6 刀有强情感锚）；交代你的审神者身份、「不对劲」的底噪、治国沙盒初识、第一位向导。
- 序章 **不剧透**：谁是谁、谁是溯行军、你能逃——这些靠后面分阶段揭。
- 刀都顶着此世身份、你认不出谁是谁 → 开局就背着一层 **「认回我的刀」** 的解密。
  - **世界线禁令**：刀在 **连接成功前**，若 **直接向任何人说出自己的真名 → 被世界线抹杀**。所以他们不能自报家门，你只能靠连接 / 解谜认回（连接机制见 §6）。

## 3. 核心循环：回合 · 动作 · 裁决

### 3.1 回合与时间
- **回合 = 可变时间步长**：每回合玩家自选推进 **日 / 月 / 季**（标准 = 月）。**行动点、事件密度、数值幅度都随步长缩放**——日 = 细控、小变化（危机 / climax 时用）；季 = 快进、大变化（平稳建设 / 养成时用）；月居中。
  - 设计意图：纯按日则回合太慢、养成推不动；可变步长让慢变量（养成、关系、世界趋势）推得动，节奏交给玩家自控。裁判按所选跨度判这一段的涟漪。
- 宏观时钟 = 季 / 月：催婚催育催立嗣、兵戈、派系情绪都在季的尺度上 tick / 升级。
- **不设硬天数上限**；四程（同心→裂→暗涌→门前，见 §11）按「进度 + 这家有多好」gate，不按时间。防拖延的软压力 = 疏于治理则熵增失序 + 派系战随进度自升级。

### 3.2 动作菜单
- 每回合一份固定行动点，花在一张 **动作菜单** 上：前朝（断案、批奏折 / 政务、任命、建设…）、后宫（巡幸、养育、求子、私通…）、外加情报窥探（调查）。每个动作动三盘。
- **情报系统（御庭番 / 隐密原型）**：可投资升级的治理子系统。**升级后解锁「窥探」**——看到 **将军不在场时的幕间**（刀的私下、派系密谋…），是主动 **搜集暗面 tell** 的工具（区别于剧情叙事里自然流露、白读到的言行 tell）。护栏：窥探 **只给片段、仍需你解读**，不把答案端上来——尤其 **「溯行军不设防时的举动」要等你确认了它的身份后才解锁**（否则窥探就把解密作弊穿了）。
- 点数稀缺 → 做不全 → **眼睛盯着江山 vs 手放在你的刀身上**，核心取舍就在这里。
- 有些危机 **强制**（不处理就吃后果）制造压力；多数动作可选，你挑着做。

### 3.3 裁决（招牌动作）
- 核心动词 **裁决（解决纷争矛盾）**，同一动作在不同尺度上重复：朝堂断案 / 调派系、后宫平争宠、继承裁嗣位，到最深一层「逃还是留」= 替自己裁的终极纷争。
- 一桩纷争 = 一幕戏（两造 / 矛盾 / 利害）。你 **亲笔裁断**（混合偏开放）；落子前系统给 **大致方向感**（能运筹），但 **幅度 / 连锁 / 会不会翻（真蝴蝶）现场长出来**。
- **每次裁决同拨三盘**：经营（国库 / 民心 / 兵戈）、权谋（派系倒向）、关系（你向着哪把刀）。三盘无法同时讨好——这就是嚼劲。
- **纷争载着暗面**：可能是某把刀做的局、溯行军的「可疑圆满」、或一记真蝴蝶的引信。**一桩纷争本身可以是 tell。**

## 4. 架构原则（大方向可行性）

> 这是决定设计形态的底层原则；**具体实现进技术文档**。本卡比作者另两张卡更 LLM 驱动。

- **引擎 = 账本**：持有并保存每一个数（国库、民心、派系、羁绊、天数、点数…），保证 **跨天不漂移**。确定性真正该守的是这里。
- **LLM = 裁判**：每次裁断，由它读着账本上的当前态、当场判这一下的涟漪（可大可小、可连锁），再把结果落回账本。**后果不预挂、当场推。**
- **作者写的是世界的「物理 / 脾性」，不是后果**：不去写死「选 A 则国库 −50」，而去写这世界的脾性（饥年加税则民心崩、武人重军功、某派记仇…），让裁判从脾性里自己收敛出后果。（= 不写死 + 「给世界观事实、让 AI 自己收敛」。）
- **蝴蝶旋钮：整体收敛、偶有真蝴蝶**。
  - 方向**默认可读**（玩家能运筹），只让**幅度与连锁**浮动；
  - 偶尔连方向都翻转 = **真蝴蝶**——而真蝴蝶是「**隐藏因果浮出水面**」、不是掷骰子，所以它事后想得通，且常常**本身就是一记 tell**（世界往反方向弯，多半有只看不见的手在拉）。
  - 响不响、翻不翻，**看游戏进度与玩家行为，不写死**。
- **软世界随便浮，硬真相不许漂**：溯行军身份、各刀立场基础线、逃的世界法则、死 = 唯一败因——这些是账本里**锁死的事实**，裁判只在其上演绎、绝不重新发明。
- 允许 **多 API 调用**（判后果 / 演剧情 / 把变化抽回账本，各走一支）。

## 5. 状态网（刻意精简——既为清楚，也为可行）

- **治国健康三件**：国库 / 民心 / 兵戈（任一触底 → 死亡风险）。
- **派系**：几个权力集团，有的由你的刀坐镇（favor 一把刀 = 抬一个派系，权谋与关系在此咬合）。
- **继承**：嗣位是否稳。
- **每把刀**：羁绊 / 觉醒 / 立场（心脏引擎，见 §6、§9、§10）。

## 6. 关系层 = 出路：重连

- **觉醒 = 重连**：认回刀、以审神者之名重结主从契、力量归位、门开。
  - ∴ **认刀、关系、脱离是同一件事**——恋爱这条高权重支线，就是出路本身。
- **连接（重连的具体动作）= 与刀交换灵力**：**早期通过体液**（亲密 / 性）——**可关、可换成皮肤接触**（照顾不同玩家）。**连接成功 → 世界线禁令解除、刀终于能道出真名 → 你认回他 + 重结契**。一条线焊死：**亲密 → 灵力交换 → 认回 + 重连 → 出路**。
- 分工再明确一次：**策略管「别死、哪些结局开放」，关系管「你重连了谁、走不走得成」。**
- **前朝、后宫两层都能推进重连**（前朝也有随机派系角色）——纯治国的玩家也会被真相找上门。
- 两层给出 **不同风味的觉醒路线**：
  - 偏前朝 = 政治化（在权斗里拼出真相）；
  - 偏后宫 = 亲密化（在私语里重接羁绊）。
  - → 喂二周目的动力。

## 7. 亲密 × 政治经济：三选一

> 把「亲密」和「政治」焊成同一套经济：你最想拉近的刀，往往也是朝堂上最顶用的政治资产。

- **纳**（正式入后宫）：拿到全部亲密，但那把刀 **真退出前朝职能**（硬牺牲）。稳，但贵——为爱拉近一把朝臣，就是在掏空你的保命班子，恋爱的风险才咬人。
- **私通**（与前朝官员暗通款曲）：保住其朝堂职能 + 照样推进重连——最划算，但落下 **把柄**：
  - 一旦败露 = 丑闻（民心 / 派系挫）；
  - 更留下一枚**长期摆在权谋盘上的棋子**：谁攥着它，谁就能在往后任意时刻拿捏你、或挑你最痛的当口引爆（一记典型的真蝴蝶）。
  - **谁替你压下丑闻、谁又让它发作，本身就是 tell。**（溯行军/留派暗中替你抹平 = 「可疑的圆满」的又一形态。）
- **纯公务**：零风险，也零链接。

## 8. 政治 · 派系 · 继承 · 养成（明面）

> 江户（德川）幕府结构为原型，架空只改性别与细节。这是**明面**的朝堂政治；刀心里的隐藏留/逃阵营见 §9。

### 8.1 推手系统（政治压力）
- 朝堂对你的常驻施压项，不理则蚀民心 / 派系 / 稳定。
- **强推手 = 立嗣三连：催婚 → 催育 → 催立嗣**（层层加码：先逼你纳妃、再逼你得子、再逼你定储）。它既是把你推进关系层的 **on-ramp**，也是 §8.4 黑暗闭环的源头。
- 其余推手：财政、边防、礼法、派系平衡…
- 暗线：催得最凶的那派朝臣，可能（自知或不自知地）在替留派 / 溯行军「把你钉死在此世」使力——一条可挖的 tell。

### 8.2 三常驻派系（+ 两个特殊力量）
每回合要权衡的常驻三家（江户原型）：
- **老中・谱代（幕阁）**：行政中枢。诉求 秩序 / 先例 / 中央集权 / 文治。杠杆 行政效率与民心；恼之 = 政务梗阻。多由前朝刀坐镇。
- **外样雄藩**：富庶善战、被中央猜防的大藩（萨摩 / 长州 / 加贺型）。诉求 自治 / 军力 / 财富 / 野心。杠杆 兵戈与财；压狠 = 谋反风险，安抚 = 纵其坐大。
- **御三家・亲藩（血）**：德川旁支，血脉与继承的看门人。诉求 正统 / 血统 /「对的」储君。杠杆 继承合法性；催立嗣的主推手。

两个不占常驻盘的特殊力量：
- **朝廷・公家（京）= 情境性法统力量**：天皇与公家，你奉其名而治。平时边缘，关键节点（册封 / 大礼 / 失德）才上场，掐你的法统（动民心与正统）。
- **大奥（内廷）= 半并入关系层**：你的妃位刀坐镇之处，内廷恩荫与储位影响。它不是独立政治盘，而是**关系层与权谋的合流点**（你宠谁、谁的孩子，直接就是政治）。**直接平衡手段**：你能 **设立御前、调动位份排名**，直接改动妃的高下——奖惩、抬派、拨动储位战的快杠杆。
- **町人・札差 / 豪商 = 财源**（非政治派系）：借贷 / 课税的活水，国库命脉。

### 8.3 连接
- **刀坐镇**：前朝刀进 老中 / 外样 / 御三家；妃位刀进大奥。favor 一把刀 ＝ 抬一派（权谋 ↔ 关系咬合）。
- **任命** ＝ 把刀 / 人放进派系位置，核心权谋动作。
- **每桩裁决的权谋盘** ＝ 你往哪派倾。
- **催立嗣 = 三方主战场**：御三家（血）× 大奥（哪个妃的孩子）× 老中（继承稳不稳）就储位开战——「为谁求子、立谁为嗣」一举牵动三派、且是恋爱、又（因孩子带不走）是逃时要撕的锚。
- **明面 vs 暗面**：这套朝堂派系（明面）与刀的留 / 逃立场（§9，暗面）分两层、但互渗——派系兴衰喂「留」的诱惑，溯行军可操弄派系，刀的派系位置是派系战的杠杆。
- **逆转上色**：将军性别玩家定、大奥为男（刀妃）；承嗣按开局开关（将军亲孕 / 刀妃孕 / 过继），御三家与大奥围着「谁来承嗣」博弈。

### 8.4 黑暗闭环（继承的双刃）
- 朝堂逼你立嗣（为江山稳）→ 孩子 / 嗣子是此世所生、**带不走** → 你为江山种下的那个嗣，正是逃跑时必须亲手撕开的锚。
- **把将军做得越称职，你就越在往这片土里扎下逃时要连根拔起的根。** 政治机器本身在替「留」给你拴绳，而绳是你为「做好本职」自己系上的。
- 可硬扛 **不立嗣**：持续吃政治罚、国更难治，但**跑得干净、没有要丢下的孩子**。又一道「治得好反而更难走」。生育可选；不生育走过继 / 册立，嗣子同为此世所生、同样带不走。

### 8.5 养成子系统（江户世子教育原型 · 有可玩深度）
- **五维**（皆有史据）：**学问**（儒学·四书五经）/ **武艺**（剑弓马）/ **礼法・教养**（书·和歌·茶·有职故实）/ **器量**（仁德·领导）/ **健康**（史上世子早夭极常见，这维是真风险）。
- **怎么养** ＝ 随时间步长推进的一串选择：
  - **傅役 / 师**：谁来教——大儒、武师，**尤其你的刀可当傅役**（养成直接接上关系与派系：刀与储君结绊、储君沾染哪派价值都是政治）。史上大奥乳母养储（春日局之于家光）＝ 这里 **妃位刀 / 大奥共育储君**。
  - 配五维的轻重；处置突发（病 / 叛学 / 惹祸…）。
- **蝴蝶**：资质 / 性情在脾性内浮动，给你惊喜或惊吓（早慧 / 体弱 / 忤逆…）。
- **通过仪礼里程碑**：七五三 → **元服**（成人礼）→ 初叙官位 → 立储，逐级把孩子推向政治成熟。
- **赌注**：**器量** → 继承稳不稳（御三家 / 老中满意，或 **御家骚动** 风险被各派利用）；同时这孩子是 **带不走的锚**（接 §8.4）。

## 9. 刀的立场：留 vs 逃（暗面）

> 区别于 §8 的明面朝堂派系：这里是每把刀**心里**的隐藏阵营。

- 每把刀有隐藏的 **留 / 逃立场**，玩家看不到，它由 **好感走向 × 世界状态 × 个刃性格** 算出——其中 **性格是决定性的那一档**：同样的爱与处境，占有欲强的刀可能至死要留你，无私的刀则甘愿放手。
- 妙处：**同一份爱因处境生相反派系**（到底往哪边倒，全看这把刀什么脾性）——
  - 深爱你 + 身处盛世 → 想留你（自私的爱）；
  - 深爱你 + 看你被这世界耗死 → 想放你走（无私的爱）。
  - 立场会随剧情 **翻面**。
- 玩家靠 **言行 tell** 推断，是一条解码成长曲线（前期瞎走 → 中期读懂 → 后期能养 / 逼一把刀转向）。
  - **「隐藏 ≠ 不可知」，tell 的可读性 = 生死线。**
- 派系战是为 **「重连」本身** 而打的：**爱 ≠ 主从契**。
  - 逃派把契 **主动递给你**（要你自由）；
  - 留派给你满怀的情、却 **扣住那最后一道契**（契一补全你就能走、就会离开它）。
  - 一把死活扣着的刀，可能在某天爱你爱到甘愿放手时，忽然松手。

## 10. 溯行军（隐藏的那一个）

- 全场 **仅一个**，是一把 **暗堕刀**（本性仍是刀男），但 **一开始就不是你的刀**——外来 / 敌方的暗堕刀，顶着某刀身份混入。
- 它要把你留下 = **溯行军使命**（坐实这条错误时间线）。起初是冷的算计，但相处中 **可能滋生一份扭曲的执念**——所以它能被感情动摇、甚至倒戈（见下）。
- **本局人设生成**：开场 roll 一个 **暗堕原因**，与 **它所顶那把刀的性格** 结合 → 产出一份 **人物性格档案**，作本局溯行军的人设。∴ 它带着所顶刀的表层性子（够像、骗得过你），又被暗堕原因拧歪——那点拧，正是 tell。
- 经营手感下它是 **诱惑者**：扮一个让国家蒸蒸日上的能臣，tell 从「可疑的灾祸」翻成 **「可疑的圆满」**。
- 它的 tell = **心有异常的惯性**：基础线高、回正慢；你推得动它（它会为感情动摇、甚至为你倒戈），但一松手又坠回「留下你」。在一片随你起伏的真心里，它是那个 **离群点**。
- 因为允许 **同名刀**（两个三日月）、且它可顶任何刀的脸（你召过的、没召过的都行）——**名字、脸、数量全失效，解读反常是揪它的唯一钥匙**。
- **能救、却带不走**：感情能渡它回头，但救了也无法带出此世——除非你自己留下。
  - 「带不走」既是它 **作恶的根**（跟不出去 → 就把你锁在这儿陪我），也是它 **救赎的根**（跟不出去 → 那你走吧，自由地走）。
  - 注意：**心可救 ≠ 契可重铸**，情感救赎与带走是两回事。
- 阴谋 / 悲剧的底色由 **暗堕原因** 定（开场 roll），再随你与它的关系演下去。

## 11. 派系战演出弧

- 它**不是预先写好的剧本，是被你的「成功」催生的**：
  - 开局其实 **没有派系战**——你刚坠进来、摇摇欲坠，所有刀利益一致（先让主公活下来）；
  - 你越把这世界治得太平、羁绊结得深、养出孩子与牵挂，就越有刀开始觉得「这样的日子，舍不得放主公走」。
  - **你越是把这里过成一个家，就越是亲手养出了不肯放你走的人。** 留派不是设定塞的反派，是你的幸福长出来的。
- **四程**（推进靠进度 + 这个家有多好，不靠天数）：
  1. **同心**（生存期）：刀都在引你活下来、看懂世界，分裂潜伏，只有「不对劲」的暗流。
  2. **裂**（觉醒 + 牵挂累积）：真相浮现 + 世界已给某些刀舍不得的东西 → 心分岔，第一批 tell 冒头。
  3. **暗涌**（权谋期）：两派不挑明、各自使力（逃派递契喂真相揭溯行军 / 留派扣契加深依赖抹丑闻攥把柄）；世上怪事开始带私心（真蝴蝶背后常是某把刀的手）；你靠 tell 慢慢 **拼出「我的刀在为我打仗」**。
  4. **门前**（收束）：一切汇于门，你显式、知情地抉择。
- **climax 响度 = emergent（看你怎么玩、随世界与玩家行为变，不写死）**：
  - 治得稳 + 读透 tell → 一场安静的 **私人清算**；
  - 治崩 + 一直被蒙 → 留派 / 溯行军借乱炸成 **逼宫**，甚至拿你的命当筹码。
- 你是被争夺的 **prize，但有 agency**（能扶持、能逼一把刀转向 = 自己下场打）；「被争夺」这件事靠 tell **慢慢渗**出来，不靠过场宣布。

## 12. 世界法则与结局

- **世界法则——逃，只能带走你带进来的**：你重连了的那些刀。
  - 门收 **羁绊**、不收花名册。没认回的、被留派扣住的、错付给溯行军的、以及孩子 / 原住民 / 溯行军——**都过不了门，留在原地**。
  - 「你只能以来时的样子离开」。逃 ≠ 胜利 = 一次切割。
- 最终的逃 / 留 = 玩家 **显式、且知情**（按下前就知道全部代价）的抉择。
- **结局哲学：除了死亡（团灭），没有「输」。** 到节点 **只如实总结、不打分**——结局画面是面镜子，不是记分牌。
- 结局空间（都只是「你的故事长成了什么样」，不分高下）：
  - **逃 / 残缺的逃**（带走重连了的，余者留下）；
  - **留**（被骗的留 vs 睁眼的留——揭没揭穿溯行军，改变它的质地，但不是一胜一负）；
  - **盲眠**（永不觉醒——更像「拒绝想起」而非「没机会」）；
  - **渡回一把暗堕刀**（救了它，但只能在「留」里与它相守）；
  - **团灭 = 死亡 = 唯一的输**。

- **真爱戳痛你，讨好你的在害你**：逼你看清牢笼的才是真心，完美顺心的可能是溯行军。
- **你的幸福，养出了不肯放你走的人。**
- **做好本职，就是在扎下逃时要连根拔起的根。**
- **你能摸透明面的「物理」，却永远解不尽这世界**——暗面随时给你一记。

## 14. 机制细化 ① · 脊柱（裁决引擎 · 账本 · 回合经济）

> 把 §3（核心循环）/ §4（架构原则）/ §5（状态网）从「脾性层」落到「机制层」：引擎与裁判怎么分工、一次裁决怎么跑、账本里到底存什么、一个回合买到多少。仍不锁数值 / prompt / 代码——这是「大方向 → 技术文档」之间的细化层。

### 14.1 引擎 ↔ 裁判边界：**账本 + 软护栏**

- **引擎 = 账本 + 软护栏**：存每一个数 + 守锁定事实 + **按步长夹单步幅度**（日 = 小步 / 月 = 中 / 季 = 大步，对 |Δ| 的夹子）+ **持有并「上膛」隐藏因果**（见 14.4）。
- **裁判 = 方向与内容的所有者**；**脾性留在 prompt**，引擎不算后果、不算方向。
- **为什么这么切**：
  - **收敛由构造保证** = 脾性在 prompt + 「方向默认可读」指令 + 引擎的幅度夹子；
  - **真蝴蝶由「上膛」门控** = 只在引擎记录了隐藏因果之处才准翻向。
  - ∴ §4 的「方向默认可读、只让幅度与连锁浮动」+「真蝴蝶 = 隐藏因果浮出、不是掷骰子」**落到了机制上**，而非全靠 prompt 自觉。
- 取舍：比「纯账本（全靠 prompt 收敛）」多两件轻确定性活，换来收敛与蝴蝶可保证；比「硬方向（引擎算方向）」少一层确定性，保住「脾性在 prompt、更 LLM 驱动」。

### 14.2 一次裁决的管线

**出幕 → [玩家] → 裁决（单 call）→ 引擎落账。**

1. **出幕**（某动作触发一桩纷争时）：裁判读账本 + 脾性 → 生成纷争（两造 / 矛盾 / 利害）+ **2–4 个立场锚** + **每锚一行定性方向感**（如「老中 ↗ 外样 ↘ 民心略沉」——*只*给方向，不给幅度 / 连锁 / 翻不翻）。方向感由本步顺产、挂在锚上，不另起 call。
2. **[玩家]**：focus 锚看方向感 → **选一个锚 / 改写 / 自己写**（= §3.3「混合偏开放」的落地：锚保证可读可预览，自由文本保住亲笔与开放）。
3. **裁决（单 call · verdict-first）**：裁判读 账本 + 脾性 + **当回合 ripe 的隐藏因果清单** + 玩家裁词 → **先吐一段紧凑 verdict**（命中哪条裁词 / 哪些 ripe 因果点火 / 每盘方向 / 是否真蝴蝶 + 所据 `缘起`），**再演戏**。用「先判后演」的硬顺序逼出冷判断，1 个 call 同时拿到收敛与蝴蝶完整性。
4. **引擎落账**：按步长**夹幅度** → 写三盘 → **更新因果登记**（点火的 retire、新上膛的入册）→ **校验锁定事实**。deltas 内联在 verdict 里，不另开「抽回账本」call。

- **适用范围**：这条全管线给**「裁决级」**动作；日常小动作（批奏折 / 建设）走**轻量版**——裁决级 vs 轻量的分流见 14.5。

### 14.3 账本 schema（硬 / 软切口）

**切口纪律**：只有「**载重 / 锁定 / 判生死或结局**」的进账本（**硬态**）；其余全归 AI 叙述（**软态**）——per-刀 live 态走叙述，别冻成静态副本和活值打架。

**硬态（引擎持有）**：

- **A. 治国三件**：国库 / 民心 / 兵戈。任一触底 = 死亡风险。
- **B. 派系倒向**：老中 / 外样 / 御三家 三个数。**载重**——引擎盯失衡阈值（见 14.4）。朝廷 = 情境事件（无常驻数，节点才上）；町人 = 财源并入国库；大奥 = 关系层合流（走妃位刀状态）。
- **C. 每把刀**（只存载重几样）：
  - **情（羁绊）· 数**：他多爱你。与契分存——§9 留派「满怀情、却扣最后一道契」的机械牙齿。日常关怀主推情。
  - **契（连接进度）· 数**：灵力归位进度 = 出路硬表。亲密 / 灵力交换主推契。
  - **立场 · 两个数**：轴 A 留你 ↔ 放你走、轴 B 随你 ↔ 离你（独立于情，裁判按角度叙事更新，见 §15.1 / §16）。
  - **个刃角度**（锁 · 定性，在性格档、不是数）：占有 / 无私 / 偏执守护…… §4「立场基线不许漂」锁在这层——决定立场怎么动、「留 / 逃」对这把刀长什么样。
  - **妃位 / 位份**（若入大奥）：调位份排名 = 储位战快杠杆。
  - 此世身份、当下情绪、在不在场 → 归 AI 叙述，不进账本。
- **D. 立场 = 独立存储值（非派生）**：轴 A / 轴 B 各一个数，**裁判按个刃角度 + 经历叙事性更新**（不是情的公式派生——高情可朝任何方向爆发）。引擎读存值出一致 tell（见 §15.1）。
- **E. 溯行军（隐藏）**：顶谁的脸（锁）+ 暗堕原因（锁）+ 执念 / 动摇值（可被你推 → 倒戈）。
- **F. 继承 / 储位**：嗣位稳定度 +（若生育）养成五维 + 储君人选。
- **G. 重连账**：每把刀 认回 flag + 重结契 flag。**门只收重结契的**（§12）。
- **H. 上膛因果登记**：一串 latent-cause 对象（见 14.4）。
- **I. 机务**：日期 / 步长 / 行动点 / 四程阶段 / 推手压力档。
- **J. 史册 / 大事记**：**锁定的既成事实**（储君被毒杀、私通败露、溯行军身份被揭、某刀倒戈…）。一旦写入 = 不许漂的真相，裁判只能在其上演绎、不能重新发明。

**软态（不进账本，AI 叙述）**：场景细节、刀的情绪与在场、对话、世界 texture、纷争具体内容…… §4「软世界随便浮」。

**立场模型 = 角度（锁）+ 立场（独立存储 · 两轴）**（解决「倒戈」与「每把刀角度不一样」的自洽）：

- **个刃角度**（锁 · 定性）：天生秉性、看待你与自由的角度，永不漂（§4）。它**只着色、不算数**——决定立场怎么动、各刀的「留 / 逃」具体长什么样。
- **立场 · 两轴 · 独立存储**：轴 A 留你 ↔ 放你走、轴 B 随你 ↔ 离你。**裁判按角度 + 经历叙事性挪**（非情的公式派生）；存值不自己抖，**决定性转向 = 裁判挪值 + 记一条史册**（不需单独的「锚」字段）。

### 14.4 隐藏因果与真蝴蝶

**latent-cause 对象（精简）**：`id` / **`缘起`**（谁因为什么埋下——「事后想得通」的弹药，点火时裁判必 cite）/ **`持有者`**（谁能引爆：某刀 / 溯行军 / 某派 / 世界——§7 的拿捏方，也是 tell 指向）/ **`触发条件`**（何时成熟，引擎可机判）/ **`域 + 倾向`**（大致推哪盘往哪偏）/ **`tell?`**（有无可读迹象 + 窥探能否揭片段）/ **`状态`**（armed → ripe → fired / defused / expired）。

**生命周期**：

1. **上膛**：四源——裁决后果（私通 → 把柄）/ **引擎失衡阈值**（见下）/ 溯行军行动 / 史册事件（毒杀储君 → 御三家记仇）。
2. **成熟**：引擎每回合查 `触发条件`，够了标 **ripe**，在 出幕 / 裁决 call 里把 ripe 子集递给裁判。
3. **点火**：裁判在某裁决**择机**引爆 ripe 因果 → 真蝴蝶（翻向 / 连锁）+ 写一条史册 + 该因果 retire / 转化。**引擎只供清单、不自己点火；裁判点火必把 `缘起` 落进 verdict**（强制可追溯）。「挑你最痛的当口引爆」（§7）= 引擎判 ripe（可响窗口）、裁判攥着等够痛的裁决才放：*时机*归裁判、*能不能响*归引擎。
4. **化解 / 过期**：玩家用**世界动作**间接拆（压丑闻 / 安抚记仇派 / 揭穿溯行军）→ retire 不引爆；部分因果 expire 变馊（防登记表无限膨胀）。

**因果默认隐藏**：玩家**不**看「因果面板」，只能靠 tell 推、靠窥探揭片段、靠世界动作化解——因果是 tell 与蝴蝶的源头，摆明即穿帮（同 §3.2 给窥探设的护栏）。

**史册（J）↔ 上膛（H）= 过去 ↔ 未来一对**：因果点火 → 写史册；史册 → 可上膛新因果。

**失衡监测（B 的牙）**：引擎确定性盯三派平衡，越界 → 上膛因果 / 起推手 / 挂常驻罚，裁判演何时咬下来：

- **外样坐大**（过高）→「割据 / 谋反」上膛；**压得太狠**（过低）→「逼反」上膛。
- **老中疏远**（过低）→ 政务梗阻：国库 / 民心 挂效率常驻罚。
- **御三家疏远**（过低）→「继承合法性危机 / 御家骚动」上膛。
- **任一派独大且有刀坐镇** → 喂留派（那把刀有了留你的本钱与理由，§8.3）。

### 14.5 回合经济

把 §3.1 / §3.2 的「可变步长 + 固定行动点 + 江山 vs 刀身」落成机制。

- **步长缩放 = 缩放「颗粒度」，不缩放行动点（AP）**：无论日 / 月 / 季，每回合**同一份**稀缺 AP；变的是**每点 AP 买到的动作颗粒**——日 =「批这一道奏折 / 见这一面」，季 =「这一季倾力练储君武艺 / 经营对外样的攻势」。同步随步长放大的是 **幅度、背景事件密度、慢变量推进**（季一跳，养成 / 关系 / 世界趋势动一大截；日一步几乎不动）。
  - 这正解 §3.1 立的题（「纯按日则养成推不动」）：要快进慢变量走季，要精细操盘走日。
  - **取舍在每个尺度都成立**：AP 每回合有界 → 永远做不全。**日 = 精度 / 低波动**（危机逐拍操盘），**季 = 吞吐 / 大跨度**（平稳期快进）。
  - **节奏是玩家的旋钮、不是陷阱**：走日纯粹「慢」（多回合换一点日历），想推养成 / 关系自然会切季。真正的防拖延（疏于治理则熵增 + 派系战随进度自升级，§3.1）管的是「不作为」，与步长正交。
- **一个共享 AP 池**（非前朝 / 后宫分账）：前朝动作与后宫动作**抢同一池点**——「眼睛盯江山 vs 手放刀身上」（§3.2）于是**字面成立**，而非两条独立预算各做各的。
- **裁决级 vs 轻量 · 分流**：
  - **裁决级**（断案 / 平争宠 / 立嗣裁定 / 重大任命 / 私通抉择）走 14.2 全管线。
  - **轻量**（批奏折 / 小建设 / 巡幸 / 调查）走快结算：单段叙述 + 小确定性 Δ，或按季打包（「这一季巡幸三次」）。
  - **升格**：裁判可把轻量动作**临时升格**为裁决——当它撞上一枚 ripe 因果时（§3.3「一桩纷争本身可以是 tell」：一次例行断案，原来是溯行军的局）。省成本之余，留出「特别时刻自己长出来」的空间。

## 15. 机制细化 ② · 心脏（tell · 留逃 · 溯行军）

> 把 §9（刀的立场 · 暗面）/ §10（溯行军）/ §3.2（情报 · 窥探）从「脾性层」落到「机制层」。**核心难题一句话:隐藏态怎么稳定地浮成「可读、却不穿帮」的 tell**——太明显 = 解密被穿，太隐 = 玩家瞎走一整局。

**核心主意:三样东西共用一台「tell 引擎」。** 留逃立场、隐藏因果、溯行军，玩家都靠**同一条 tell 流**去读;而**溯行军不是独立机制，是这台引擎下的一个「反常离群点」**（它的立场被锁死、不随经历挪）。这条想通，心脏就不散。

### 15.1 立场:独立存储 · 裁判按角度更新 · 引擎读值出 tell

- **立场 = 两个独立存储值**（轴 A 留你 ↔ 放你走 / 轴 B 随你 ↔ 离你，见 §14.3），**和情没有公式关系**——高情可朝任何方向爆发（占有式地留、无私地放、舍命放你走、死活要同行……），**每把刀的角度不一样**，公式收不住。
- **裁判叙事性挪立场**：按个刃角度（锁）+ 经历（你怎么待它、它目睹了什么、成长弧）推——不是每场重算，是**存着的值、只在真出事时被挪**。
- **引擎读存值出 tell**：→ **存值 = tell 一致**（玩家学得会那套暗语，§9 解码曲线）；**裁判挪值 = 每把刀角度不同**（收住复杂）。一致与复杂同时拿到。
  - 关键分工：**写**立场的是裁判（叙事），**读**立场出 tell 的是引擎——引擎不拿公式算立场（那样收不住「每人角度不一样」）。

### 15.2 tell 管线:引擎吐结构化指令，裁判只渲染措辞

- 每场景引擎产 **0–2 条结构化 tell 指令** = `{目标 · 漏哪个隐藏事实 · 强度档 · 建议通道(微表情/话/行动) · 禁止线}`;**裁判只把指令渲成 in-fiction 散文**（一瞬迟疑 / 一个眼神 / 一句过满的话），**不做校准判断、绝不点名机制、绝不报数、绝不超本档清晰度**。= 把 LLM 最不擅长的「拿捏多明显」从它手里收归引擎离散档，只留它最擅长的「这一档怎么写好看」。
- **分档范式库（作者写 · 内容侧）**:每档配一组渲染范式（faint = 半秒停顿 / 移开视线 / 答得太顺;clear = 当面矛盾 / 失言），裁判 **照库改写、不凭空发明校准** → faint 真 faint、clear 真 clear、跨局一致。
- **一致性锚（治长局漂移）**:每次渲染喂入 **该刀近期 tell 历史 + 溯行军锁死惯性参数**，保证「这把刀一路的暗语」连贯、溯行军的反常每 call 都锚在锁死档案。
- **强度三档 faint / moderate / clear，默认压在 faint–moderate**:clear 很稀，只在大节点放（某刀逼近转折、一枚因果刚点火）。→ 日常底噪**微妙**（不穿帮），偶尔才给清楚信号。强度由 **漏压**（隐藏态多强:强烈倾向 / 极端反常漏得多）× **场景钩子**（没有自然时机就不硬塞）共同定。
- **永不端答案**:引擎不 surface「X 是溯行军」「倾向 = 逃 73」，只给行为，玩家自己拼。
- **诚实、不掺纯噪音**:tell 不说谎——溯行军模仿的是表层，它的反常是**真** tell;「可疑圆满」是**真**信号。难在解读，不在辨真假。

### 15.3 溯行军:同一台引擎下的反常离群点

- **开场生成**:6 刀定后引擎 roll ——① 顶谁的脸（你的刀 / 某刀的同名分身 / 你没召过的刀皆可）② 暗堕原因（池中抽）。**顶刀表层性格 + 暗堕原因的「拧」→ 锁定人物档案**（schema-E:顶谁 + 原因锁，执念 / 动摇可变）。
- **它的 tell = 立场的反常惯性**:别的刀立场**随经历自然挪**，裁判却按它锁死的暗堕原因把它**轴 A 钉在「留你」不动**——情冷了、世界崩了、换别的刀早松口，它**黏在留你不放**（轴 B 对它无意义，本就带不走）。一片随境流动的真心里，**它是那个不跟着动的点**。→ 这正是为什么**名字、脸、数量全失效，解读反常是唯一钥匙**（§10）:你读的不是脸，是**反常的惯性**。
- **暗堕原因 = 那个「拧」的具体形状**（兼悲剧底色）:「曾被主公抛弃」→ 留-基线病态高、回正 ≈ 0（说什么都不放手）;「错失救主」→ 过度操控式的圆满（替你摆平一切 = 可疑圆满）。原因锁死 → 反常一致 → tell 可学。
- **执念 / 动摇可推**:你能用情动摇它（执念 ↓）、甚至令其倒戈;但一松手又坠回（回正慢 = 它的本性）。**能救心、不能重铸契、带不走**（§10 照旧）。

### 15.4 tell 怎么到你手里:叙事自然流露（免费）+ 情报主动探（§17.3）

- **行为 tell 在裁判叙事里自然流露（免费 · 非机制）**:刀 / 因果的 tell 本就织进在场叙事（一瞬迟疑 / 过满的话 / 可疑的圆满），faint–moderate。这不是你「发动」的系统，是角色本来就这么演（§9）——**零投资玩家照样读得到，解密对谁都不断**。
- **情报系统 = 唯一的「主动探」工具（花钱 · roll · 受派系掣肘 · 详 §17.3）**:想看**将军不在场的幕间**（不在场刀的私下、派系密谋、因果持有者的盘算）→ 经御庭番主动探。**给更多数据点、仍是生料不端答案**（§3.2）;护栏:**溯行军不设防的举动须先坐实其身份才解锁**（§16.4）。
- **可读性的地板 = 数据量**（§17.3）:不擅读叙事 raw 的玩家砸钱探 → 幕间碎片堆够 → 靠量三角测;擅长的直接读叙事。两头都接得住——**但谁都拿不到明牌**。

### 15.5 tell ↔ 因果（挂上 §14.4）

- tell 流不只漏立场，也漏**因果**（§14.4 的 `tell?` + `持有者`）:一枚因果上膛 → 漏出指向其持有者的 tell（留派刀攥着把柄的眼神 / 溯行军埋局后的可疑圆满）。
- 于是「**读一把刀的心**」与「**察觉一桩阴谋**」是**同一动作的两面**，都在这条流里。

> **「从怀疑到坐实溯行军身份」的具体动作** 接在 ③ 关系层（§16.4）。

### 15.6 读 tell = 可验证的累积循环（容错设计）

> 设计**不指望任何单条 tell 一击命中**（LLM 做不到稳定的「微妙却可感」）;改让 **系统容忍单条模糊**，把读 tell 做成过程。这条是 tell 解密的承重保险，与 §15.2「校准归引擎」一体两面。

- **假设 → 检验 → 收敛**:读到疑点 → **试探 / 对质 / 窥探定向**（§16.4 / §17.3 已有手段）→ 裁判 in-fiction 确认或反证（那把刀的反应）→ 玩家三角测、逐步坐实。∴ 没有哪条 tell 必须完美,读 tell 是 **累积过程**、非一次性知觉检定。
- **早期斜坡容错**（§17.4c）:同心期 tell 稀又淡、漏了无损——玩家有一整段学「暗语」的跑道;到暗涌才真需读准,那时玩家 + 循环都校准好了。
- **可读性地板 = 数据量**（§15.4）:读不来 raw 的砸钱探,碎片堆够也能三角测——又一层容错。
- **设计的唯一义务**:tell 在玩家「有看」时 **可读 + 稳定地投进他正读的叙事**;玩家读不读、读不读得出,是他的故事（§12 不打分），**设计不逼他看、也不设兜底金手指**。

## 16. 机制细化 ③ · 关系层 = 重连（出路本身）

> 把 §6（关系层 = 出路）/ §7（亲密 × 政治经济）/ §12（世界法则）落到机制。**重连一把刀 = 三道 gate（认回 → 契满 → 重结契），第三道由立场把守**——你整局读立场、推它转向，就在这第三道兑现成「带不带得走」。

### 16.1 两个量 + 两轴立场，各司其职

- **情（羁绊）**：他多爱你。→ 推动连接 / 亲密（从而推契）、开他的口（情高 → tell 更多更清、认回更易）、是极端表达的燃料。
- **契（连接进度）**：灵力归位进度 = **出路的硬表**。
- **立场（轴 A 留你 ↔ 放你走 / 轴 B 随你 ↔ 离你）**：和情无公式关系（§15.1）。
- 一句话：**情 是爱、契 是出路表、立场 把守「契能不能兑现成重结契」。**

### 16.2 怎么推

- **情** 两层都涨：后宫（照顾 / 亲密）、前朝（共事 / 患难）。
- **契** 走「连接（灵力交换）」：默认经亲密 / 体液，**可关、可换皮肤接触**——这是**呈现层开关（NSFW on/off），机制不变**（契照推，只换怎么演）。连接-act 走 §7 的 纳 / 私通 动作、吃共享 AP（私通连接同时上膛一枚把柄，§7）。
- **立场** 不靠玩家直接推，**靠裁判按角度 + 你的所作所为叙事性挪**（§15.1）。

### 16.3 三道 gate

1. **认回（flag）= 你确认他真身**。三路：
   - **连接-reveal**：契推到「连接成功」阈值 → 世界线禁令解除 → 他终于敢道真名（§6）→ 认回。
   - **解谜**：tell 读够 → 你自己点破（§2）→ 认回。
   - **真身自白（牺牲）**：刀主动认下抹杀的代价、说出真身来唤你（§2 禁令的双面）——**强制认回，但刀就此消失、带不走**。裁判叙事自然触发，引擎只记后果（刀除名 + 认回 + 史册），不写专门逻辑。
2. **契满 = 灵力可重铸线**。不到线，再情愿也铸不成。
3. **重结契并随你回家 = 认回 ✓ ∧ 契满 ✓ ∧ 轴 A 放你走 ∧ 轴 B 随你走**（§12 门只收重结契的）：
   - **轴 A 留你 → 扣契**：情再深、契再满也扣着（铸成你就能走、就会离开它，§9）。**翻它 = 裁判把它轴 A 叙事性挪向放你走**（看这把刀的角度 + 它经历了什么；可能是它目睹这世界在耗死你、爱你爱到甘愿放手）。
   - **轴 A 放你走 ∧ 轴 B 离你 → 递契不同行**：它给你自由、自己留下 / 牺牲 → **不在你门口的花名册上**，又一种「余者留下」（§12）。

### 16.4 溯行军在关系层的特例（收掉延后线）

- **坐实身份**：反常 tell（离群点，§15.3）读够 → 你能对怀疑下手（试探 / 对质）→ 坐实它是冒名者。坐实后，§3.2「窥探它不设防的举动」才解锁。
- **能救心、不能重结契**：它不是你的刀，「契可重铸」不适用（§10 心可救 ≠ 契可重铸）→ **带不走**。能救心（执念 ↓、甚至倒戈），只能在「留」里与它相守。
- **doppelganger 陷阱 + 一记冷 tell**：它顶名顶脸甚至同名，连接-reveal 还能报个**假真名**骗你认回 → reveal 对真刀可靠（禁令逼真话）、对它会撒谎，**解读反常才是唯一可靠钥匙**（§10）。副作用信号：你情契拉满、立场也该翻了、却**怎么都重结契不成** =「这把有鬼」的一记 tell（与「留派死扣」同形，仍需辨）。

### 16.5 两层 = 两种觉醒风味（喂二周目）

- **偏后宫 = 亲密化**：连接快、契涨快，认回多走 reveal。
- **偏前朝 = 政治化**：情 / 认回在权斗患难里拼；契靠你愿不愿让盟友贴近（哪怕只皮肤接触）。**全程冷面 = 真相照样找上门（认得回），但带得走的人少（契难满）= 孤独的逃**——「恋爱 = 出路」的代价。

## 17. 机制细化 ④ · 子系统

> 把已定的脊柱（§14）/ 心脏（§15）/ 关系层（§16）套进四个子系统、补各自判定规则——多是**套用、非新发明**。顺序（作者拍板）：**立嗣三连推手 → 养成五维 → 情报 / 窥探 → 派系战四程**。仍不锁数值 / prompt / 代码。

### 17.0 开放总则 · 推手非死轨（贯穿全子系统 · 锐化 §4 / §14.1 / §14.4）

任何 **推手 / 政治矛盾都不是逼你就范的单轨**——多条合法策略都能扛过或绕过，裁判按 **现实逻辑** 收敛后果（§4 不写死）。三类典型，机制上 **都已兼容、不必加东西**：

1. **创意 defer / transfer**：玩家自由文本裁（以边患未平为由奏请缓议 / 挑起外样事端盖过催声…）→ 裁判按脾性判成 / 半成 + 代价。**压力是个账本值，引擎 tick（默认漂移）与裁判 verdict（你做了什么）都能写它**，引擎照常按步长夹幅度。**你逃不掉、只能 defer / transfer**：被按下的压力不蒸发，裁判可上膛成日后 **回潮的因果**（一记将来的蝴蝶）。创意在「种类」上不设限，后果被账本追踪 = 软护栏。
2. **治理出色**：**失衡监测的牙挂在民心 / 稳定 / 派系上，不挂在「你照办了没」**→ 三件丰厚者吸收得起常驻罚、喂饱相关派系者 tick 变慢（强君付得起代价，弱君被磨垮）。
3. **平衡钳制**：派系倒向是独立数，制衡是裁判可读的现实态 → milestone 上可判主推派「孤掌难鸣」、不照办也收得了场。

∴ 每个推手都是 **真两难**（各路都合法、§12 不打分），而非死轨。

### 17.1 立嗣三连推手（催婚 → 催育 → 催立嗣）

> 把 §8.1（推手）/ §8.4（黑暗闭环）落到机制，并以此立起 **通用推手模板**。

**a. 结构 = 单一压力线 + 三阶指针（非三个并存推手）**

- 账本只存 `立嗣压力（数）+ 当前阶（催婚 / 催育 / 催立嗣）`。三连是 **顺序闸**（没纳妃无从催育、没子无从催立储），任一时刻只一阶「活」。
- 迈过一阶（纳妃 → 产子 → 定储）→ 压力 **转形** 进下一阶、基线更高、不清零。
- **通用推手模板** = `{压力值 · tick 速率 · feeder · 不理的罚 · 可选阶段指针}`。立嗣是唯一带三阶指针者；财政 / 边防 / 礼法 / 派系平衡（§8.1）= 此模板的 **扁平版**（无阶段指针）。一次定义、五推手通吃。

**b. 行为 = 复用 §14.4 失衡监测（不另起惩罚逻辑）**

- **tick（引擎 · 确定性 · 每季）**：压力按 `当前阶权重 × 无嗣停滞时长 × 御三家 mood` 爬——越后的阶越陡、御三家恼则更快；纯账本记账，同派系 drift。
- **feeder = 当前阶 + 无嗣停滞 + 御三家 mood** 三者。刻意 **不** 把盛世 / 危机做硬 feeder（切口纪律）——世道交给裁判叙事或上膛因果的 context。御三家由此 **双重把守继承**：喂压力 + §14.4「疏远 → 上膛继承危机」。
- **罚 = 失衡监测两道牙，且牙挂在「民心 / 稳定 / 派系」上、不挂在「纳妃了没」**（这是 §17.0「多路可度」的机械保证）：
  - 软阈 → 挂 **常驻罚**（民心 / 稳定 drain，随档放大，同老中疏远挂效率罚）。
  - 硬阈 → **上膛因果**「御家骚动 / 继承合法性危机」，裁判择机点火、必 cite 缘起。

**c. 升格 = 每一阶的「迈过去」都是裁决级 milestone**

- 背景里只 tick（轻量底噪）；办「**纳妃 / 求子 / 定储**」那一下升格成 **裁决级 milestone**（走 §14.2 全管线）：出幕（御三家奏请纳妃 / 两候选妃各带派系背书的夺嫡局）→ 玩家亲笔裁 → verdict → 落账（三盘涟漪）。
- 最深一阶 **催立储 = 三方主战场**（御三家 × 大奥 × 老中，§8.3），大型 set-piece。
- = §14.5「轻量临时升格为裁决」的标准用法。

**d. 黑暗闭环落账 = 不设专门「锚」字段**

- 孩子作为 **储君人选** 进 schema F（养成 / 继承本就要此字段），零新增机制。
- 「带不走」**不是 flag**，是 §12 世界法则在 **结局结算** 套用：门只收重结契的刀，孩子是 native（§8.4 此世所生者皆 native：刀妃孕 / 过继 / 册立一律算）→ 落进「余者留下」。
- 黑暗闭环咬人靠 **沉没真投入**（AP / 养成的季 / 继承稳定度 / 刀傅役结的羁绊 §8.5）在门前被迫割舍——不是合成锚值。立储当下 **给继承稳定度 ↑**（治国 / 派系收益）= 诱你扎根那口糖，正中 §13「做好本职 = 扎下要拔的根」。

**e. 两只角（接 §17.0：立嗣推手 = 真两难，非死轨）**

- **入局**（纳妃 / 立储）→ 称职（治国稳 / 继承稳）**但扎下带不走的锚**（§8.4）。
- **凭本事顶住**（治理出色 + 平衡钳制）→ **跑得干净、没要丢的孩子**，但更难更孤独（放掉继承稳定度这块缓冲 + 时时维持制衡 + §16.5「孤独的逃」）：
  - **治理出色** = 三件丰厚吸收常驻 drain；喂饱御三家（让正统 / 血统 / 先例、礼遇之）→ tick 变慢。
  - **平衡钳制** = 催立储 milestone 上派系制衡让裁判判御三家孤掌难鸣 → 不纳妃也收场（孤立退步 / 转过继 / 摄政案）。
- 两条都合法、§12 不打分。

### 17.2 养成五维（储君 = 会留在此世的人）

> 把 §8.5「有深度养成子系统」落到机制。核心：储君不是数值块，是个 **有自己资质、政治倾向与三观的人**，由多源合熬而成——而你亲手塑的这个人，正是门前要丢下的那个（§13）。

**a. 五维 = 全进账本 · 双层载重**

- 学问 / 武艺 / 礼法・教养 / 器量 / 健康 五维 **全是账本数**（养成跨十几季，裁判记不住会漂——持久态归账本 §14）。
- **硬载重（失衡监测的牙挂这俩）**：器量 → 继承稳定度 / 御家骚动风险；健康 → 夭折 = 真生死（史上世子早夭极常见）。
- **软载重（养成轻数，无直接生死）**：学问 / 武艺 / 礼法 → 喂器量 + gate 仪礼里程碑 + 给储君上色。

**b. 资质 = 逐维天赋向量 · 出生锁定**

- 出生蝴蝶掷出 **逐维天赋档**（武艺天纵 / 学问平平 / 体弱…），**锁**（同个刃角度的定性锁），只随大事件微漂；挂 schema F。
- 它是 **五维成长的修正器**：高天赋维投入少长得快，低天赋维要么重砸要么认命 → 养成的真取舍：**扬长 vs 补短**。裁判拿它上色（「殿下弓马天纵，然不喜诗书」）。

**c. 储君人格 = 政治倾向（账本）+ 三观（软态）**（照搬刀模型 §14.3）

- **政治倾向 = 账本载重 dial**（挂 schema F）：内化了哪派价值（老中秩序 / 外样军力 / 御三家正统）的 lean + 强度。**有牙**：立储时御三家掂量正统（§8.3 三方主战场更咬）；成年倾向与你的当权派冲突 → 上膛 御家骚动 / 储君党争（§8.5）。
- **三观 / 理念 / 家庭观 / 人生观 …「等等角度」= 软态人格，裁判养**（定性、不入账）：**账本存不下「等等」，裁判才给得起无穷角度**。裁判把储君当连贯的人来长，锚在 五维 + 倾向 + 资质 / 性情 + **史册锁死的里程碑事实** 上。

**d. 人格由五源合熬（将军本人是大头）**

- 裁判按脾性把五源熬成 倾向 dial + 三观软态，**非任何单项决定**（§4 给脾性让裁判收敛、天然防 min-max）：
  1. **天生资质 / 性情**（蝴蝶；有的天生忤逆，教不回）；
  2. **生育者**（哪个妃 / 刀妃所出，其性情 / 派系烙印）；
  3. **教养者**（日常带大的人 = 日浸最重，§8.5 春日局之于家光）；
  4. **傅役 / 师**（授业者，多源之一、**非决定项**）；
  5. **★将军本人的行动**——储君看你 **怎么真的治国**（裁决史、你怎么待刀 / 民 / 派系）；**你做出来的价值观比你教的更入骨**。
- **影响源绝大多数是普通 NPC**（§1：乳母 / 大儒 / 妃多为此世普通人，软态）。**当某影响源恰是你的刀 / 溯行军（认出或没认出）= 特例，叠两层**：① 关系 / 派系耦合（刀傅役结绊、储君沾其派系，§8.5）；② **一条 tell vector**——储君的发展会 **漏** 出影响者真身（没识破的刀傅役经教学漏 tell；溯行军假扮教养者把储君往「此世真好，留下」推 = 养成版「可疑的圆满」）→ 养成由此也是解密的一面（挂 §15）。

**e. 怎么养 = 裁决级 vs 轻量映射（套 §14.5）**

- **轻量季推进**：配傅役 / 教养者、配五维轻重——「这一季倾力练武艺」打包季结算，小确定性 Δ（按资质修正）。
- **裁决级 spike**：**突发**（病 / 叛学 / 惹祸）与 **仪礼里程碑** 升格全管线（§14.2），玩家亲笔裁、三盘涟漪。

**f. 仪礼里程碑 = 逐级推向政治成熟（§8.5）**

- 七五三 → **元服（成人礼，三观锁档最重的节点）** → 初叙官位 → 立储（= §17.1 催立储 milestone，三方主战场）。
- 每道是一桩 **里程碑裁决**，把储君往政治舞台推一级；元服前后是裁判把成形三观写进史册的关键关。

### 17.3 情报 · 窥探（御庭番 = 要养、要守、靠 roll 的眼睛）

> 把 §3.2 / §15.4 的情报系统落到机制。一句话定性：**不是花钱开图的金手指，是一台 政治化、要主动维持、按距离付费、靠 roll 出货 的探测系统。**

**a. 双成本 = 资本升级（养网）+ 每探付费（按距离 × 影响力）**

- **资本投资（养御庭番）**：花国库 / 建设升「**情报等级**」（账本值，挂 schema I），解锁窥探的 **广度上限**（能探多远的层）+ **抬 roll 的胜率与上限**（见 b）。= §3.2「可投资升级的治理子系统」。
- **每次发动窥探付国库钱 · 三因子定价**：**看后宫极廉**（将军自家控制圈内）→ **越出你的控制范围越贵**（探远藩密幕最烧）→ **且反比于将军影响力**（你越强、网越深探得越省；越弱每探越贵）。∴ 情报吃 **国库**（与升级 / 建设 / 养成 / 赈灾抢钱），不走 AP 线——是你下的一道 **令**（御庭番去办），稀缺在钱、不在行动点。

**b. 非金手指 · 探测靠参数 roll**

- **允许探，但「能否成 + 成多少」= 参数 roll**：输入 = 情报等级 / 主导度（见 c）/ 目标距离 / 将军影响力 / 目标是否设防…→ 输出 = 失败（空 / 碎渣）→ 部分 → 丰收（多条 tell 指令）。**花了钱也可能扑空**；等级 / 主导抬的是分布，不是保底。
- roll 成功后，引擎照 §15.2 发 tell 指令（强度仍 faint–moderate、clear 稀有）、裁判演成幕间行为。**等级 / roll 调的是 reach + volume，绝不调清晰度**——堆满也只是生料消防栓，不是判决。

**c. 派系化 + 主导地位（要主动维持，否则真但片面）**

- 情报网 **本身嵌在派系政治里**，有一个 **主导度**（账本：谁把着御庭番——你、还是某派渗透了它）。受派系影响——得宠 / 坐大的派系会伸手进来。
- **你得主动维持主导**（任命亲信 / 整肃 / 持续供养 = 持续成本 / 偶发裁决）；**松手 → 被某派把持 → 情报变「真但片面」**：给的全是真的（tell 不说谎 §15.2），但是 **那派筛过的子集**——盲区 = 它自家的勾当，内容偏向服务它的议程。
- **片面本身是一记 meta-tell**（「为何我的御庭番从不报外样？」）；且 **可被留派 / 溯行军反用**（把持你的眼睛、专挑喂你往「留」想 = 情报版「可疑的圆满」）。

**d. 广度三档 + 两道护栏**

- **广度（资本解锁，档数示意）**：Lv1 个人私下幕间 → Lv2 派系密谋 / 多方的局（漏因果 tell §15.5）→ Lv3 最深盘算（谁替留派 / 溯行军使力、把柄何时引爆 §7）。
- **护栏 1 · 生料不端答案**：见 b——清晰度恒压、花钱买不穿，你永远自己拼。
- **护栏 2 · 溯行军不设防幕间锁在「坐实身份」后**（§3.2 / §16.4）：坐实前探一个 **恰是溯行军** 的目标，只得和探任何刀一样的暧昧反常 tell（它藏得好）；**先靠反常 tell 坐实，才解锁看它卸防的盘算**——靠解密解锁、花钱买不穿。

**e. 飞轮**

- 影响力 ↑ → 探得更省更广 + roll 更旺 → 看得更透 → 治得更稳 → 影响力再 ↑；反之弱势 + 失主导 → 又贵又片面又常扑空 → 越来越瞎。**情报忠实地奖励强势、惩罚失控，绝不当谁的金手指。**

### 17.4 派系战四程（暗线主线 · 与明线治国平行）

> 把 §11（派系战弧）/ §3.1（不按天数）落到机制。它是罩住前三个子系统的 **容器弧**，也是全卡的 **暗线主线**。

**a. 明 / 暗双线平行（全卡结构）**

- **明线 = 治国**（看得见的经营 / 权谋 dashboard：断案 / 派系 / 养成 / 立嗣）= **地基**（§0：决定别死、哪些结局开放）。
- **暗线 = 四程**（觉醒 / 重连 / 留逃 / 溯行军）= **主线**（§0：剧情真正发生处——重连了谁、出不出得去、最终成为谁）。
- **焊点 = §3.3「每次裁决同拨三盘」**：每个治国动作（明）同时喂暗线——经营 / 权谋 = 明面涟漪，关系 / 立场 / 觉醒 = 暗线推进。∴ 纯治国玩家也被真相找上门（§6），沉迷关系也得治国才不死。
- **暗线不给进度条**：明线有 dashboard，暗线只能靠 **tell / 基调** 感觉（推到哪程是你读出来的、非 UI 告知）——同删读心、tell 不端答案一脉（§15）。

**b. 推进 = 引擎读派生复合量过阈（不存新值 · 不按天数 · 治崩停不下）**

- 只存 **当前阶 enum**（schema I）；三个 gate 量全从已有账本 **派生**（切口纪律）：
  - **觉醒进度** = Σ认回 + Σ契（schema G/C）——推进弧的 **主引擎**。
  - **家好度** = 治国三件 + Σ情 + 孩子 / 牵挂 + 派系稳——① 给「裂」当 **软门**（低门槛 · 中盘几乎必过）；② **持续喂留派立场漂**（§8.3 / §11 越治好越有刀舍不得 → 裁判挪轴 A）。
  - **立场分化** = 刀们轴 A / 轴 B 离散度（schema C）——「裂 → 暗涌」的门。
- **四程 gate**（示意 · 数值进技术文档）：
  1. **同心** = 默认开局态（world 摇晃、刀利益一致）。
  2. **同心 → 裂** = 觉醒进度过 T1（真相浮现）∧ 家好度 ≥ 最低牵挂。
  3. **裂 → 暗涌** = 立场分化过阈（两派成形）+ 暗面因果 active。
  4. **暗涌 → 门前** = 重连逼近门（够认回 ∧ 契满）或玩家显式推向门。
- **弧单调向前、治崩停不下**：疏于治理 → 熵增触发失衡因果 + 觉醒 / 真相照揭 → 弧照推；治崩只 **改 climax 形状**（见 d），不踩刹车 = §3.1 / §11「防拖延 = 派系战随进度自升级、不按天数」。

**c. 每过一程 = 暗线往明线渗得更响（escalating）**

- **同心**：暗线蛰伏，几乎不渗（只「不对劲」底噪）。tell 密度最低、因果多在 armed、推手刚起（催婚）。
- **裂**：暗线开始渗——第一批 tell 冒头、因果开始 ripe、推手加码（催育 / 立嗣）；明线事件开始暗藏私心。
- **暗涌**：暗线借明线机器开打——怪事带私心（真蝴蝶背后是刀的手）、tell 密度高、因果点火率峰值、两派拿派系 / 把柄 / 丑闻当兵器。**明线成了暗线的战场。**
- 机制落点：**当前阶 = 引擎调 tell 密度 / 因果上膛 · 点火率 / 推手强度 的总旋钮**（阶越深、三者越烈）。

**d. 门前 climax 响度 = 暗线撞破明线撞得多狠（emergent · 不写死）**

- 响度 = 读 **(明线治国 state) × (暗线掌握度)**：
  - **治稳 + 读透 tell + 重连理顺 → 私人清算**：两线都驾驭住，暗线无需暴烈撞破，门前是一场安静的私人了断。
  - **治崩 + 被蒙 + 重连烂尾 → 逼宫**：暗线借崩塌的明线炸开，留派 / 溯行军拿公共危机 + 你的命当筹码。
- 引擎读 治国三件 ↓ + 失主导 / 被蒙 ↑ + 重连未完 ↑ + 失衡因果堆积 ↑ → 越高越往逼宫；裁判收敛具体形状（§11）。你是被争夺的 prize 但有 agency（扶持 / 逼一把刀转向 = 自己下场打）。
