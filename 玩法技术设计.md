## 0. 一句话哲学

**引擎管数，裁判管戏，作者写脾性。**

- **引擎（账本）** 持有所有数值，保证它们跨天不漂、生死线说一不二。
- **裁判（LLM）** 读着账本的当前状态，当场推断这一下的后果、演出剧情。
- **作者** 不写死后果，只写这个世界的「脾性」（物理规则），让裁判从脾性里自己收敛出后果。

这三句话决定了下面的一切。

## 1. 引擎 vs 裁判：谁负责什么

传统做法是把后果写死：「选 A → 国库 −50」。本作不这么干——那样既僵硬、又永远写不完。改成分工：

- **引擎只做三件确定性的事**：① 存每个数（国库 / 民心 / 兵戈 / 派系 / 每把刀的羁绊…）；② 守住锁死的真相（谁是溯行军、世界法则、死＝唯一败因）；③ 按时间步长夹住单步幅度（日＝小变化 / 季＝大变化）。
- **裁判（LLM）负责方向与内容**：这次裁决往哪偏、涟漪多大、会不会连锁翻盘、剧情怎么演。
- **作者写的是世界观事实**：「饥年加税则民心崩」「武人重军功」「某派记仇」——给裁判一套脾性，让它自己算出合理后果。

好处：后果永远「当场新鲜地长出来」、贴合上下文，又因为数值锁在引擎里、不会跨天漂移。

## 2. 账本里存什么、什么交给 AI

切口纪律一句话：**只有「承重 / 锁死 / 决定生死或结局」的东西进账本（硬态），其余全交给 AI 叙述（软态）。**

- **硬态（引擎存）**：治国三件（国库 / 民心 / 兵戈）、三派倒向、每把刀的（羁绊·数 ＋ 连接进度·数 ＋ 立场两轴·数）、继承 / 储君、重连进度、隐藏因果登记、日期 / 步长 / 行动点、史册（锁死的既成事实）、基建设施等级……
- **软态（AI 演）**：场景细节、刀的情绪与在不在场、对话、世界质感、纷争的具体内容……

为什么这么切：能改写剧情的东西交给 AI 才灵活；能决定输赢的东西交给引擎才不漂。一把刀「此刻心情如何」是软态（随便演），但「他多爱你、连接到几分」是硬态（锁死）。

## 3. 一次裁决怎么跑（核心循环）

核心动词是 **裁决**——断案、调派系、平争宠、定储，直到最深的「逃还是留」。一次「裁决级」动作的管线：

1. **出幕**：裁判读账本＋脾性，生成一桩纷争（两造 / 矛盾 / 利害），给 2–4 个立场锚，每个锚标一行「大致方向感」（如「老中 ↗ 外样 ↘ 民心略沉」——只给方向，不给幅度、不剧透会不会翻盘）。
2. **玩家落子**：看方向感 → 选一个锚 / 改写它 / 干脆自己亲笔写裁词。（「混合偏开放」：锚保证可读可预览，自由文本保住亲笔与发挥。）
3. **先判后演**：裁判先吐一段紧凑的「判决」（命中哪条裁词、每盘往哪偏、是否触发真蝴蝶、依据是什么），**再演戏**。用「先判后演」的硬顺序逼出冷静判断，不让剧情带着数值跑。
4. **落账**：引擎按步长夹住幅度 → 写回三盘 → 更新隐藏因果 → 校验锁死真相 → 据新的隐藏状态生成「tell 指令」喂给演绎。

**每次裁决同拨三盘**——经营（国库 / 民心 / 兵戈）、权谋（派系倒向）、关系（你向着哪把刀）。三盘无法同时讨好，这是核心嚼劲。日常小动作（批奏折 / 巡幸）走轻量快结算，不必每次都跑全管线；但裁判可以把一个例行动作临时「**升格**」成裁决——当它撞上一枚成熟的隐藏因果时（一次例行断案，原来是溯行军的局）。

## 4. 隐藏因果 ＋ 真蝴蝶：后果怎么会「翻盘」

这是「整体收敛、偶有真蝴蝶」落地的机制。

- **隐藏因果**：引擎维护一串「上膛的因果」，每个记录——**缘起**（谁因为什么埋下的，点火时裁判必须引用，保证「事后想得通」）、**持有者**（谁能引爆：某刀 / 溯行军 / 某派 / 世界）、**触发条件**（何时成熟）、**倾向**（大致推哪盘往哪偏）、**有无 tell**、**状态**（上膛 → 成熟 → 点火 / 拆除 / 过期）。
- **从哪来**：裁决后果（私通 → 落下把柄）、引擎的失衡监测（外样坐大 → 上膛「谋反」）、溯行军的行动、史册事件（毒杀储君 → 御三家记仇）。
- **怎么爆**：引擎每回合检查哪些因果「成熟」了，把成熟清单递给裁判；裁判择机引爆 → **真蝴蝶**（方向翻转 / 连锁）＋ 写一条史册。

关键：**真蝴蝶不是掷骰子，是「隐藏因果浮出水面」**。世界往反方向弯，背后总有一只看得见缘起的手——所以它事后想得通，而且往往本身就是一记 tell。玩家看不到「因果面板」，只能靠 tell 推、靠情报揭片段、靠世界动作（压丑闻 / 安抚记仇派）去拆。

## 5. 立场与 tell：解密怎么做到「可读却不穿帮」

这是全卡最精巧的一块——隐藏信息怎么稳定地浮成「玩家读得到、却不被白送答案」的线索（**tell ＝ 角色无意流露的破绽**）。

核心主意：**三样东西共用一台「tell 引擎」**——每把刀的留 / 逃立场、隐藏因果、溯行军，玩家都靠同一条 tell 流去读。

- **立场 ＝ 两个独立存储的数**：轴 A「留你 ↔ 放你走」、轴 B「随你 ↔ 离你」。它和「爱你多少」**没有公式关系**——高情可以朝任何方向爆发（占有式地留 / 无私地放）。
- **谁来动立场**：不是引擎用公式现算，而是 **裁判按「个刃角度」（每把刀天生的秉性，锁死不变）＋ 经历，叙事性地挪动那个存值**。占有欲强的刀和无私的刀，面对同样的爱与处境会往相反方向走。
- **谁来出 tell**：**引擎读这些存值，吐出结构化的 tell 指令** ＝ ｛对谁 / 漏哪个隐藏事实 / 强度档 / 建议通道（微表情·话·动作）/ 禁止线｝。
- **裁判只负责「渲染」**：把指令写成剧情里的散文（一瞬迟疑、一个眼神、一句过满的话），**绝不点名机制、绝不报数字、绝不超过这一档的明显度**。

这么切的妙处：把 LLM 最不擅长的「拿捏多明显」从它手里收归引擎的离散档位（保证淡的真淡、明的真明、跨局一致），只留它最擅长的「这一档怎么写好看」。强度三档 **faint / moderate / clear**，日常压在 faint–moderate（微妙不穿帮），clear 很稀有、只在大节点放。

**删读心 ＋ 容错**：没有花钱开图的金手指。tell 分两路到你手里——① 剧情里**自然流露**（免费，零投资也读得到）；② 主动养**情报网**（御庭番）去探将军不在场的幕间（花钱、靠 roll、只给生料碎片）。而且设计 **不指望任何单条 tell 一击命中**：读 tell 是「假设 → 试探 / 对质 / 定向窥探 → 裁判在剧情里确认或反证 → 逐步坐实」的累积循环，系统容忍单条模糊。

## 6. 溯行军：同一套引擎下的「离群点」

溯行军（混入的伪装者）**不是一套独立机制，而是上面这台 tell 引擎下的一个反常离群点**。

- 别的刀立场随经历自然挪；它的立场被 **锁死在「留你」** 不动（它的暗堕原因决定了它回正极慢）。
- 于是在一片随你起伏的真心里，它是那个「情冷了、世界崩了、换别的刀早松口，它却始终黏着你」的点。**名字、脸、数量（可同名 / 可顶任何刀）全部失效——读懂这股反常惯性，是揪它的唯一钥匙。**
- 它的阴谋有递进弧（**渗透 → 布局 → 收网**），引擎按条件确定性推进，偏置它每一步用哪类手段——这是暗线的「升级器」。但它的动机（留你）始终不变。

## 7. 回合与节奏推进

**回合 ＝ 可变时间步长**：每回合玩家自选推进 **日 / 月 / 季**。

- 缩放的是「颗粒度」**不是行动点**：无论日月季，每回合同一份稀缺行动点；变的是每点 AP 买到的动作颗粒（日＝批这一道奏折 / 季＝这一季倾力练储君武艺）。
- 同步随步长放大的是：数值幅度、背景事件密度、慢变量推进（养成 / 关系 / 世界趋势季一跳、日几乎不动）。
- 解决的题：纯按日则养成推不动、回合太慢；可变步长让玩家自控节奏——要精细操盘走日，要快进慢变量走季。

**暗线四程（主线）：同心 → 裂 → 暗涌 → 门前**

- 推进 **按「进度 ＋ 这个家有多好」，不按天数**。三个门槛量全从已有账本派生：**觉醒进度**（认回数 ＋ 连接数）、**家好度**（治国 ＋ 羁绊 ＋ 牵挂 ＋ 派系稳）、**立场分化**（各刀立场的离散度）。
- 越往后，暗线渗进明线越响：同心期几乎只有「不对劲」的底噪；到暗涌，怪事都带私心、tell 密度最高、因果点火率峰值，明线机器成了暗线的战场。
- **弧只向前、治崩停不下**：疏于治理不会让剧情停摆，只会改变高潮的形状。这就是防拖延——不靠天数上限，靠「派系战随进度自升级」。

**节奏 ＝ 引擎软导演（全确定性，不写剧本）**

四程靠进度推、步长玩家自控，天然容易「磨」（卡住没进展）、「踩空」（高潮来早了玩家还没投入感情）、或张力一路平。引擎升级成一个 **软导演**，只用确定性手段维护一条「起—伏」曲线：

- **卡住就推**：引擎数着「距上次进展多少回合」「同类动作连击几次」，超阈就给已有旋钮（因果成熟速率 / 政治压力 / tell 密度）挂压力乘子——**让「磨」本身生出事来**，逼出移动。
- **高潮要「攒够了」才放**：过门槛除了看进度，还加一道「情感投入」条件——感情没到位，大节点就不触发，避免高潮来得太早、没分量。
- **限速 ＋ 保供**：防止节奏暴冲或饿死，程序内对推进速率限幅、对必要事件保底供给。
- **关键纪律**：这一切全落在 **确定性** 那一侧、不新增 LLM 判断——节奏太重要，不能押给「裁判临场拿捏」这种不稳定的东西。

**高潮响度 ＝ 涌现的，不写死**：同样到门前，治得稳 ＋ 读透 tell ＋ 重连理顺 → 一场安静的私人清算；治崩 ＋ 一直被蒙 ＋ 重连烂尾 → 留派 / 溯行军借公共危机炸成逼宫，甚至拿你的命当筹码。引擎读「治国状态 × 暗线掌握度」决定往哪种倒。

## 8. 数值只是脚手架：数值为正文服务

一条贯穿始终的元纪律：**所有标度、阈值、速率都是脚手架——正文（剧情）才是主题。**

- 数值的「档」代表的是 **叙事意义层级**：轻＝日常涟漪 / 中＝一幕戏 / 重＝格局震动。
- 校准目标是「故事感觉对」，不是数学平衡。
- 引擎拥有真相、AI 拥有讲法、**戏在演绎里、不在判决的数字里**。别让数值变成目的本身。

## 9. 记忆与上下文：长局不崩、秘密不泄

LLM 的上下文有限，长局会「忘」。本作的对策：

- **章回体**：把剧情切成一「章」（以剧情节点为界、不死绑回合），章末做一次总结、压缩进滚动记忆，并生成回目。
- **人物卡**：每个重要人物一份滚动更新的总结（七把刀恒有，保暗线连续性）。
- **关注（hook）**：玩家手动关注重要人物（上限约 10、几乎免费）＝ 把注意力放谁身上，被关注者更多织进正文。**身份盲选**——你不知道谁是谁、很可能关注到普通人，这本身就是解密玩法；它也给上下文注入封了顶，长局 token 近乎恒定。
- **秘密只走裁决那一路**：演绎用的那次 call 只拿到「tell 指令」，真正的秘密（谁是溯行军、隐藏因果原文）只有裁决那一步看得到——**秘密永不进「总结 / 记忆 / 向量库」**，从根上防泄题。

## 10. 结局结算：镜子，不是记分牌

- 结算规则是确定性的：逃，门 **只收你重连了的刀**（认回 ＋ 连接满 ＋ 它的立场放你走 ＋ 愿随你走）；没认回的、被留派扣住的、错付给溯行军的、孩子、原住民——都留在原地。「你只能以来时的样子离开。」
- **除了死亡（团灭），没有「输」**。到结局只如实总结、不打分。结局空间（逃 / 残缺的逃 / 被骗的留 / 睁眼的留 / 永不觉醒的盲眠 / 渡回一把暗堕刀）只是「你的故事长成了什么样」，不分高下——它是一面镜子，不是记分牌。
